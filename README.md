# Sobre mim

### Olá, bem-vindo(a)!

Meu nome é Jorge Silva, bacharel em Tecnologia da Informação e Engenharia da Computação pela [Universidade Virtual do Estado de São Paulo (UNIVESP)](https://univesp.br/) e faço parte da equipe da [TENDÊNCIAS CONSULTORIA](https://tendencias.com.br/), no setor de Tecnologia, Informação e Dados. Trabalho na interseção entre Engenharia de Software e Ciência de Dados, desenvolvendo soluções de dados end-to-end, da ingestão ao consumo via APIs. Desenvolvo pipelines ETL para séries temporais em alta escala, utilizando arquitetura orientada a microsserviços para garantir dados confiáveis, rastreáveis e prontos para uso analítico.

### Responsabilidades

* Desenvolvimento de pipelines ETL distribuídas com Python e Apache Spark
* Modelagem, versionamento e transformação de dados em Delta Lake
* Integração de múltiplas fontes (APIs REST, bancos de dados e web scraping)
* Criação de APIs e microsserviços para disponibilização dos dados tratados
* Automação, agendamento e monitoramento de pipelines no Databricks

### Impacto

Processamento automatizado de indicadores econômicos, séries temporais e dados operacionais, reduzindo o tempo de disponibilização e habilitando decisões de negócio em tempo real.
